Mozilla Location Service Leaderboard

Contributing to mapping the world!

Overview

The Mozilla Location Service Leaderboard tracks the contributions made to the Mozilla Location Service. This leaderboard shows the contributions of individual community members, who are using one of the client applications and contribute back data to us - we call this activity stumbling. Joining this competition is optional and you can contribute data anonymously. You only show up on the leaderboard if you have collected at least ten reports.

How do I contribute?

Mozilla Stumbler is an open-source wireless network scanner which collects GPS, cellular and wireless network metadata for our crowd-sourced location database.

You can find Mozilla Stumbler on Github to contribute to its development.
